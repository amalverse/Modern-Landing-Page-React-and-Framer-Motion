# Modern Landing Page

# live - https://modern-landing-page-react-and-frame.vercel.app/

Modern Landing Page is a clean, responsive React + Vite template designed to showcase products, services, or portfolios with modern visual polish. It includes modular components like a `Navbar`, `Hero`, `Features`, `Pricing`, `Testimonials`, and `Footer`, and uses small motion helpers for subtle animations.

Built for quick customization and fast development feedback, this project is ideal for startups, freelancers, and marketing sites. Customize styles in `App.css` and `index.css`, and modify or extend components under `src/components` to match your brand.

# Key highlights:

- Lightweight Vite setup for fast dev builds
- Modular React components for easy reuse
- Responsive design and accessibility-minded structure

# Tech used:

- React
- Vite
- JavaScript (ES6+ / JSX)
- HTML & CSS
- Tailwind CSS
- Framer Motion
- Small motion helper: `src/utils/motion.js`
